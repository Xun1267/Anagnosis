# CONSENSUS_Lexicon.md

## 最终共识确认

### 明确的需求描述

**项目目标**: 开发一款AI驱动的英文原著阅读器MVP版本，支持文件导入、智能解析和本地数据管理。

**核心功能**:
1. **文件管理**: 支持EPUB、TXT文件导入，生成个人书架
2. **智能解析**: 划词/划句触发AI解析，毛玻璃弹窗展示结果
3. **阅读体验**: 目录导航、进度保存、个性化设置
4. **准入控制**: 邀请码"welcome"验证机制

### 技术实现方案

#### 前端架构
- **框架**: React 18 + TypeScript
- **构建工具**: Vite
- **状态管理**: React Context API
- **样式方案**: CSS Modules + Tailwind CSS
- **路由**: React Router v6

#### 核心依赖库

#### 数据存储策略

- **目标浏览器**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+

- **文件大小限制**: 单个文件最大50MB
