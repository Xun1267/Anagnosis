# 笔记功能使用指南

## 功能概述

笔记功能是 Anagnosis 的核心特性之一，允许用户在阅读过程中创建、管理和查看智能笔记。该功能与 AI 分析功能深度集成，提供了丰富的笔记类型和便捷的管理方式。

## 主要功能

### 1. 文本高亮和笔记创建

#### 通过文本选择创建笔记
1. 在阅读页面选择任意文本
2. 点击弹出的 AI 分析按钮
3. 选择分析类型（分析、翻译、总结、解释）
4. 点击"开始分析"获取 AI 结果
5. 点击"保存笔记"将分析结果保存为笔记

#### 笔记类型映射
- **分析模式** → 词汇笔记 (vocabulary)
- **翻译模式** → 词汇笔记 (vocabulary)
- **总结模式** → 语义笔记 (semantic)
- **解释模式** → 词汇笔记 (vocabulary)

### 2. 笔记高亮显示

#### 高亮样式
- **词汇笔记**: 蓝色实线下划线
- **语法笔记**: 绿色虚线下划线
- **文化笔记**: 紫色点线下划线
- **语义笔记**: 橙色波浪线下划线

#### 交互功能
- **点击高亮**: 打开笔记侧栏并显示笔记详情
- **鼠标悬停**: 显示笔记预览（可选）
- **右键菜单**: 快速操作选项（可选）

### 3. 笔记侧栏管理

#### 笔记详情查看
- 显示笔记标题、内容和创建时间
- 显示笔记类型和标签
- 显示原始选中文本和位置信息

#### 笔记编辑功能
- 编辑笔记标题和用户备注
- 添加和管理标签
- 保存修改内容

#### 笔记列表管理
- 查看当前文档的所有笔记
- 按类型、时间等条件筛选
- 搜索笔记内容
- 删除不需要的笔记

## 使用场景

### 1. 语言学习
- 选择生词进行翻译和分析
- 保存语法解释和用法说明
- 记录文化背景知识
- 整理重要概念和总结

### 2. 学术研究
- 标记重要段落和概念
- 保存 AI 分析的深度解读
- 建立知识点之间的关联
- 快速回顾和复习要点

### 3. 日常阅读
- 记录有趣的观点和想法
- 保存值得回味的句子
- 建立个人知识库
- 提升阅读理解效果

- 精确记录文本选择位置

- 流畅的用户交互体验

## 快捷操作

### 键盘快捷键
- `Ctrl/Cmd + N`: 创建新笔记
- `Ctrl/Cmd + S`: 保存当前笔记
- `Escape`: 关闭笔记侧栏
- `Ctrl/Cmd + F`: 搜索笔记

### 鼠标操作
- **单击高亮**: 查看笔记详情
- **双击高亮**: 快速编辑笔记
- **右键高亮**: 显示上下文菜单

## 注意事项

### 1. 浏览器兼容性
- 推荐使用现代浏览器（Chrome 90+、Firefox 88+、Safari 14+）
- 需要支持 IndexedDB 和现代 JavaScript 特性

### 2. 数据安全
- 笔记数据存储在本地浏览器中
- 建议定期导出备份重要笔记
- 清除浏览器数据会删除所有笔记

### 3. 性能建议
- 单个文档建议不超过 1000 个笔记
- 定期清理不需要的笔记
- 避免创建过长的笔记内容

## 故障排除

### 常见问题

#### 1. 高亮显示异常
- 刷新页面重新加载
- 检查文档结构是否发生变化
- 清除浏览器缓存

#### 2. 笔记保存失败
- 检查浏览器存储空间
- 确认 IndexedDB 功能正常
- 尝试重新创建笔记

#### 3. 侧栏无法打开
- 检查是否有 JavaScript 错误
- 确认笔记数据完整性
- 重启应用程序
